# DATASET
We used 1,340 normal chest X-ray images.

Dataset can be downloaded in [[link]](https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia).



# Training
Input Size : 512x512

Latent Vector Size : 512**2

Batch Size : 64

Number of epoch : 200
